# C# University Portal & Chat 

backend written in C#, Frontend written in Typescript / React

This C# / React application powers a university portal and chat system, providing a backend API for a React front-end.

### What it does:

- Parses and transforms data from university portal pages.

- Manages users, universities, subjects, schedules, lecturers, and syllabuses with full CRUD operations.

- Generates and checks mock exams to help students prepare efficiently.

- Supports AI-driven chat using LLMs, including real-time message streaming.

- Connects seamlessly with a React front-end for interactive and dynamic user experiences.
